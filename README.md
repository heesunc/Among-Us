# 개요
대유행을 탄 게임 '어몽어스' 클론코딩

# 프로젝트를 하면서 배운 점
어몽어스 게임을 즐기는 플레이어로 그치지 않고,
클론코딩으로 미니게임들을 개발하며 어떻게 작동하는지 공부할 수 있었습니다.

# 실행방법
appbuild 폴더에 apk파일을 zip파일로 묶어두었습니다.
다운 받아 zip 해제 후 apk 파일을 앱 에뮬레이터나 스마트폰에서 작동시키면 됩니다.
